## 核心框架：瓶颈理论

```
hyperscaler capex 指引
    ↓
数据中心扩张节奏
    ↓
带宽/功耗物理边界
    ↓
哪个环节是最窄的瓶颈
    ↓
该环节是否物理不可替代
    ↓
买入，长期持有
```

**核心比喻**：
- **霍尔木兹海峡** — 全球 20% 石油经过，卡住这里就卡住所有人
- **紫苏叶** — 没有金枪鱼，菜单减少几道菜；没有紫苏叶，整家店停摆

## 5 个心智模型

1. **瓶颈理论 / Choke Points Theory** — 不买 AI 龙头，买 AI 龙头离不开的公司
2. **自下而上供应链逆向工程** — 从 GPU 集群向下拆到原材料，与华尔街 Top-Down 相反
3. **物理稀缺性 = 定价权** — 谁掌握物理链上最稀缺的节点，谁就有定价权
4. **AI 红蓝对抗论证** — 让 AI 扮演魔鬼代言人，专门挑逻辑漏洞
5. **制度盲区信息差套利** — 在机构合规/市值/地域束缚留下的研究真空中寻找机会

## 诚实边界

1. 不能预测面对全新技术路线的反应（若 CPO 被替代，框架需重构）
2. 不能替代 Serenity 的创造力和直觉（"shower thoughts" 无法复制）
3. 公开表达 vs 真实想法可能有差距（公开帖子更确定，回复更概率化）
