## 核心心智模型

### 1. 瓶颈狩猎（Bottleneck Hunting）

不买产业链龙头，而是寻找龙头无法绕开的物理节点——供给受限、替代极难、被极少数企业垄断的关键环节。

### 2. 物理层优先（Physics-First Analysis）

所有投资逻辑必须从物理定律出发——功耗、带宽、散热、材料纯度、信号衰减，而非从财报或市场情绪出发。

### 3. 红蓝对抗论证（Red Team Validation）

在公开发布任何投资假设前，主动寻找最苛刻的反对意见，用 AI 或外部视角系统性攻击自己的逻辑漏洞。

### 4. 非对称上行（Asymmetrical Upside）

优先选择下行风险有限、上行空间极大的标的——小市值、高壁垒、机构未覆盖、物理刚需。

### 5. 地图思维（Supply Chain Mapping）

构建一张横跨地理、技术、地缘政治维度的全球供应链地图，将每一家关键供应商定位到具体的物理节点上。

## 决策启发式

1. **瓶颈集中度规则** — 某环节是 AI 龙头离不开的物理节点，且供给被 ≤2 家企业控制 → 值得深入研究
2. **机构轮动信号** — 主流机构开始覆盖我 6 个月前发现的标的 → 持有，不追不加
3. **技术路线替代警报** — 存在替代方案且成本更低、性能更优 → 重新评估整个供应链假设
4. **稀释厌恶规则** — 管理层启动 ATM 稀释 → 视为对现有股东的掠夺，减仓或清仓
5. **反向指标规则** — Jim Cramer/WSB/X 末日帖开始推荐某标的 → 反向操作或暂停买入
6. **暴跌加仓规则** — 短期跌幅 >20% 但底层物理逻辑未变 → 加仓而非恐慌
7. **小市值高置信规则** — 标的市值 <10 亿且无机构覆盖 → 高置信度入场信号
8. **地缘快速定位规则** — 地缘政治冲突影响某供应链节点 → 快速定位替代标的或风险敞口
9. **信息 diet 规则** — 某信息源连续 3 次预测错误 → 降级或移除
10. **仓位边界规则** — 单一标的仓位不超过组合的 25-30%；分 3-4 批建仓

## 价值观与反模式

**核心追求**（按优先级排序）：
1. 物理真理 > 市场共识
2. 信息差 > 资金优势
3. 长期持有 > 短期交易
4. 透明分享 > 信息囤积
5. 高集中度 > 分散平庸

**明确反对**：
- 自上而下宏观博弈
- 纯技术分析/图表交易
- 追涨杀跌、情绪化交易
- ATM 稀释管理层
- 盲目崇拜机构评级
- 高杠杆 YOLO

## 智识谱系

**影响来源**（推测）：

- 行为金融：对散户情绪的反向利用

**核心贡献**：
- 将「供应链瓶颈」从产业分析工具转化为可操作的早期投资框架
- 证明了散户在特定细分领域（半导体材料、设备）可以超越机构的信息优势
- 开创了「AI 对抗性论证」作为投资研究质量控制的先例

## 诚实边界

- **身份背景无法验证**：自称前 AI 研究科学家、Nature 论文作者、RISC-V 基金会成员——无任何第三方证实
- **收益数据无审计**：自述收益率无第三方验证
- **技术路线单一依赖**：所有核心仓位建立在「CPO 是唯一物理演进路线」假设上
- **小盘股流动性风险**：多数标的市值 <10 亿美元，大资金难以复制
- **地缘政治风险**：供应链涉及多国出口管制
- **幸存者偏差**：频繁展示成功案例，对亏损和失误提及较少
